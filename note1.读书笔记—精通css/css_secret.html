三、凸显不同类型的链接

问题：不同类型链接相同的样式，使用户混淆

目标：区分不同类型的链接，分别加icon或用不同的样式来显示

原理：属性选择器允许根据特定属性是否存在 或 属性值 来查找元素

(1) [attr^=val] (eg、[href^="http"]) —— 寻找以val开头的所有链接

(2) [attr$=val] (eg、[href$=".doc"]) —— 寻找以“.doc”结尾的文件
